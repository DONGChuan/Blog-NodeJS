# MiniBlog-NodeJS

A mini blog to learn NodeJS. It has the following features:

* Register
* Login/logout
* Add post (title, tag, content)
* List posts
* Show post details

From this demo, I learnt mainly the following points:

* express routes
* how to use mongoose
* how to use session
* how to use express view engine
* ejs
